# FinGuide AI

## SBI Hackathon 2026

### Problem Statement
Digital Adoption

## Overview

FinGuide AI is an Agentic AI-powered banking adoption assistant designed to improve customer engagement and increase adoption of SBI digital services.

The platform acts as a personalized financial companion that understands customer behavior, recommends relevant banking products, guides onboarding processes, and improves financial literacy through intelligent interactions.

## Key Features

- Personalized Product Recommendations
- Autonomous Onboarding Assistant
- Multilingual Support
- Financial Literacy Guidance
- Smart Reminder System
- Digital Adoption Score
- Customer Journey Tracking
